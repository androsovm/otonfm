# Oton.FM v2 -- UX/UI Дизайн-спецификация

> Полная спецификация визуального дизайна, компонентов, анимаций и взаимодействий приложения Oton.FM.
> Документ предназначен для разработчиков: каждый экран и взаимодействие описаны без двусмысленности.

---

## 1. Философия дизайна

### Концепция
Oton.FM -- современное радио-приложение с тёмным, погружающим интерфейсом, вдохновлённым Spotify и Apple Music, но с уникальной якутской идентичностью. Интерфейс минималистичен: один экран, одна кнопка, одна цель -- слушать радио.

### Принципы
1. **Погружение** -- фон приложения живёт и дышит вместе с музыкой. Градиенты Якутии создают атмосферу, а адаптивный цвет обложки связывает визуал с контентом.
2. **Минимализм** -- ничего лишнего. Обложка, название трека, кнопка Play. Всё.
3. **Якутская идентичность** -- 25 тематических градиентов, вдохновлённых природой и культурой Якутии. Текст "Холбонуу..." на якутском языке.
4. **Тактильность** -- haptic feedback на каждое значимое действие. Приложение ощущается физически.
5. **Стабильность** -- фиксированные позиции элементов, никаких "прыжков" интерфейса при смене состояний.

### Эмоциональная палитра
- **Тепло** -- даже в тёмной теме интерфейс ощущается уютным благодаря тёплым акцентным цветам
- **Спокойствие** -- плавные анимации, медленная смена градиентов (10 секунд)
- **Надёжность** -- стабильный layout, предсказуемое поведение

## 3. Библиотека компонентов

### 3.1 PlayButton

Круглая кнопка воспроизведения, центральный элемент управления.

**Размеры**: 64x64pt круг, белый фон, тёмная иконка.

#### Состояния

| Состояние | Иконка | Доп. элементы | Описание |
|-----------|--------|---------------|----------|
| **Idle (Play)** | `play.fill` 30pt bold, цвет `#181818`, offset x=+2pt | - | Поток остановлен, готов к запуску |
| **Playing (Pause)** | `pause.fill` 30pt bold, цвет `#181818`, offset x=0 | - | Поток воспроизводится |
| **Buffering** | Скрыта | `ProgressView` circular, tint `#181818`, scale 1.2 | Буферизация при isBuffering && isPlaying |
| **Pressed** | Как текущее | scaleEffect = 0.9 | Пользователь нажал кнопку |

#### Анимации

| Переход | Тип | Длительность |
|---------|-----|-------------|
| Pressed -> Released | `.easeOut` | 0.2s |
| Play -> Pause icon | Мгновенный swap | - |

#### Haptic Feedback

| Событие | Паттерн |
|---------|---------|
| Нажатие (touch down) | `.light` impact |
| Отпускание (touch up) | `.light` impact |
| Действие (play/pause) | Complex: `.heavy` + `.light` через 0.1s |

```
+------------------+     +------------------+     +------------------+
|                  |     |                  |     |                  |
|    (  >  )       | --> |    ( ... )       | --> |    ( || )        |
|                  |     |                  |     |                  |
|  Idle (Play)     |     |  Buffering       |     |  Playing (Pause) |
+------------------+     +------------------+     +------------------+
```

### 3.2 ArtworkCard

Карточка обложки альбома -- центральный визуальный элемент.

**Размеры**: screenWidth * 0.85 (ширина и высота)
**Скругление**: RoundedRectangle(cornerRadius: 10)
**Тень**: averageColor @ 0.6 opacity, radius 25, y=10

#### Состояния

| Состояние | Содержимое | Фон экрана | Пульсация |
|-----------|------------|-----------|-----------|
| **Default (no artwork)** | defaultArtwork из ассетов | Анимированные градиенты Якутии | Нет (только при isPlaying) |
| **Station Logo** | stationLogo из ассетов | Градиенты или адаптивный (зависит от hasLoadedRealArtworkOnce) | При isPlaying |
| **Real Artwork** | Загруженная обложка трека | Адаптивный (averageColor -> black) | При isPlaying |
| **Error** | Остаётся предыдущая обложка | Без изменений | Нет |

#### Анимации

| Анимация | Тип | Длительность | Условие |
|----------|-----|-------------|---------|
| Пульсация | scaleEffect 1.0 <-> 1.02 | 1.5s repeatForever | isPlaying && pulsateAnimation |
| Смена обложки | `.easeInOut` на artworkId | 0.6s | Новый artworkId |

### 3.3 TrackInfoLabel

Текстовый блок с названием трека или статусом.

**Расположение**: горизонтальный padding = screenWidth * 0.075, offset y = -40pt
**Высота фрейма**: 60pt, alignment: .leading
**Шрифт**: system 22pt bold white, максимум 2 строки

#### Состояния

| Состояние | Содержимое | Приоритет |
|-----------|------------|-----------|
| **Connecting** | "Холбонуу" + AnimatedDots | Высший (isConnecting == true) |
| **Playing with track** | Название трека (до 2 строк) | Средний (isPlaying && !trackTitle.isEmpty) |
| **Idle / No track** | "OTON FM" | Низший (всё остальное) |

#### Переходы между состояниями

```
transition: .opacity.combined(with: .move(edge: .bottom))
animation: .easeInOut(duration: 0.5)
```

Анимация привязана к трём значениям: `isConnecting`, `trackTitle`, `isPlaying`.

### 3.4 GradientBackground

Полноэкранный фоновый градиент, заполняющий весь экран (.ignoresSafeArea).

#### Режимы

| Режим | Когда | Описание |
|-------|-------|----------|
| **Animated Yakutia** | isDefaultArtworkShown == true | Плавная интерполяция между 25 градиентами |
| **Artwork-based** | isDefaultArtworkShown == false | LinearGradient от averageColor (@ 0.8) к #121212 |

#### Параметры анимации (Yakutia mode)

| Параметр | Значение |
|----------|---------|
| Интервал смены | 10.0 секунд |
| Длительность перехода | 3.0 секунды |
| Тип перехода | `.easeInOut` |
| Метод | RGBA-интерполяция (lerp) |
| Стартовый градиент | Случайный при каждом новом показе |
| Порядок | Последовательный (index + 1) % count |

#### Параметры анимации (Artwork mode)

| Параметр | Значение |
|----------|---------|
| Анимация перехода | `.easeInOut(duration: 0.8)` |
| Привязка | artworkId |

- Padding внешний горизонтальный: 40pt

#### Элементы

| Элемент | Стиль |
|---------|-------|

**Представление**: `.sheet`

#### Автоматический показ

Оверлей, появляющийся после успешной покупки.

**Z-Index**: 2
**Transition**: `.scale.combined(with: .opacity)`
**Фон**: чёрный @ 70%, cornerRadius 20
**Тень**: чёрная @ 50%, radius 20, y=10

#### Элементы

| Элемент | Стиль |
|---------|-------|
| Иконка | `checkmark.circle.fill`, 80x80, цвет accent.primary |
| Текст (якутский) | "Komoнг иhин баргfа махтал!", 24pt bold, белый |
| Текст (русский) | "Спасибо за покупку!", 16pt, серый |
| Кнопка | "Продолжить", 16pt bold, чёрный на accent.primary, Capsule |

### 3.8 SplashScreen

Загрузочный экран приложения.

**Длительность**: 2.0 секунды
**Переход**: fade-out 0.5s `.easeInOut`

#### Слои (снизу вверх)

1. LinearGradient: #121212 -> #191414, top -> bottom
2. RadialGradient: #D00000 @ 30%, center, startRadius=50, endRadius=180, пульсация
3. Логотип "otonLogo-Light": 100x100pt, scale анимация

#### Анимации

| Элемент | Свойство | От | До | Длительность | Тип |
|---------|----------|----|----|-------------|-----|
| RadialGradient | scaleEffect | 1.0 | 1.2 | 1.0s | repeatForever(autoreverses) |
| RadialGradient | opacity | 0.3 | 0.7 | 1.0s | repeatForever(autoreverses) |
| Logo | scaleEffect | 0.9 | 1.1 | 1.2s | repeatCount(1, autoreverses: false) |
| Logo | opacity | 0.8 | 1.0 | 1.2s | repeatCount(1, autoreverses: false) |

### 3.9 AnimatedDots

Компонент анимированных точек для текста "Холбонуу...".

**Количество точек**: 3
**Таймер**: 0.4 секунды
**Spacing**: 2pt между точками

#### Цикл состояний

```
Tick 0: [   ] [   ] [   ]   -- все скрыты
Tick 1: [ . ] [   ] [   ]   -- 1 видна
Tick 2: [ . ] [ . ] [   ]   -- 2 видны
Tick 3: [ . ] [ . ] [ . ]   -- все видны
Tick 4: [   ] [   ] [   ]   -- сброс
```

**Анимация opacity**: `.easeIn`
**Шрифт**: наследуется от родителя (system 22pt bold)

## 5. Анимации и переходы

### 5.1 Каталог анимаций

#### Смена обложки

| Параметр | Значение |
|----------|---------|
| Тип | Crossfade по artworkId |
| Длительность | 0.6s |
| Easing | `.easeInOut` |
| Триггер | Изменение artworkId (UUID) |

#### Градиенты Якутии

| Параметр | Значение |
|----------|---------|
| Интервал смены | 10.0s (Timer) |
| Длительность перехода | 3.0s |
| Easing | `.easeInOut` |
| Метод | RGBA-интерполяция (lerp) между top/bottom цветами |
| Стартовый градиент | random(0..<25) |
| Направление | Последовательное (index + 1) % 25 |
| Управление | Таймер запускается/останавливается при isDefaultArtworkShown |

**Алгоритм перехода**:
```
1. Timer fires (каждые 10s)
2. gradientTransition анимируется 0.0 -> 1.0 за 3s (.easeInOut)
3. Для каждого кадра:
   topColor = lerp(current.top, next.top, gradientTransition)
   bottomColor = lerp(current.bottom, next.bottom, gradientTransition)
4. Через 3s: currentIndex = nextIndex, nextIndex++, gradientTransition = 0.0
```

### 8.2 Паттерн Complex Haptic (Play/Pause)

```
Время:  0ms          100ms
        |             |
        [HEAVY]       [LIGHT]
        impact        impact
```

Двухступенчатый удар: сильный "клик" с последующим лёгким "отзвуком" через 100мс. Создаёт ощущение физической кнопки.

### 8.3 Альтернативный Complex Haptic (CHHapticEngine, ContentView)

```
Время:  0ms          100ms        300ms
        |             |            |
        [TRANSIENT]   [CONTINUOUS-------]
        i=1.0 s=0.7   i=0.5 s=0.5 dur=0.2s
```

Резкий удар с плавным затуханием. Этот паттерн определён в ContentView, но в v1 не используется напрямую (используется простая версия из PlayerControlsView).

**Рекомендация для v2**: Использовать единый haptic-сервис, избегая дублирования реализации.

## 12. Спецификация переходов между состояниями

### 12.1 Матрица состояний PlayerState -> UI

| PlayerState | Фон | Обложка | Текст | Кнопка | Error | Пульсация |
|-------------|-----|---------|-------|--------|-------|-----------|
| `.stopped` | Градиенты Якутии | Default | "OTON FM" | Play | Нет | Нет |
| `.connecting` | Градиенты Якутии | Default/текущая | "Холбонуу..." | Spinner | Нет | Нет |
| `.playing` | Адаптивный/Якутия | Реальная/default | Название трека | Pause | Нет | Да |
| `.paused` | Последний | Последняя | Последнее название | Play | Нет | Нет |
| `.buffering` | Текущий | Текущая | Текущее | Spinner | Нет | Нет |
| `.error(*)` | Текущий | Текущая | Текущее | Play | ErrorView | Нет |

### 12.2 Переходы (что анимируется при переходе)

```
.stopped -> .connecting:
  - Текст: "OTON FM" -> "Холбонуу..." (opacity + move, 0.5s)
  - Кнопка: Play -> Spinner (мгновенно)
  - Haptic: complex (heavy + light)

.connecting -> .playing:
  - Текст: "Холбонуу..." -> название трека (opacity + move, 0.5s)
  - Кнопка: Spinner -> Pause (мгновенно)
  - Обложка: crossfade 0.6s
  - Фон: градиенты -> адаптивный (0.8s)
  - Пульсация: начинается
  - Haptic: .medium (при получении метаданных)

.playing -> .stopped (через pause):
  - Текст: название -> "OTON FM" (через playing -> stopped cycle)
  - Кнопка: Pause -> Play
  - Пульсация: прекращается
  - Haptic: complex (heavy + light)

.playing -> .error:
  - ErrorView: появляется поверх
  - Пульсация: прекращается
  - Кнопка: Pause -> Play

.error -> .stopped (retry):
  - ErrorView: исчезает
  - Начинается новый цикл подключения
```
